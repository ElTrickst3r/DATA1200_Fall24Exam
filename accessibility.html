Manual Measures and Testing for Accessibility

The project is built on a traditional body-style with header, main and footer. We have used semantic HTML with typical elements such as strong, aside, ul, li and iframe in addition to standard heading conventions.

With regards to images we have ensured that all images have alt-attributes with decriptive text for people with visual impairments. We have done this in a similar fashion for links as well by adding titles within the a-tags.

Regarding manual testing, we have used Edge, Firefox, Chrome for PC, as well as Chrome for iOS and Android. We have also involved four family members and acquaintances as testers.
